# nuggie Grabber
Grab things from the Web, Programs, and more!
Written in Windows Batch Script, VB Script and PowerShell Script

# Tutorial
- Open the nuggieGrabber.exe file that you downloaded,
- Type in "help" to see all commands,
- Choose a command to try,
- For example: "http request"
- Type in your URL that you want the HTML data from,
- The data is saved in nuggie Grabber\output\web_html_32452.html
